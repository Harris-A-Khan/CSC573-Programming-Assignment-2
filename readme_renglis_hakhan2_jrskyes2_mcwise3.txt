PREREQUISITES: 

1) Python installed on test machine 
2) Pip installed on test machine


INSTALLATION:

1) Clone the repository to your local machine
2) Navigate to the root directory of the repository
3) Run the following command to install the required packages:

pip install scapy or pip3 install scapy

4) Run the following command to run the program:

sudo python3 sniffer_renglis_hakhan2_jrskyes2_mcwise3.py

The program is running!
